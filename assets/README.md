# GlobalDeets Assets

This directory contains icons and images for the GlobalDeets landing page.

## Required Assets

To complete the setup, you'll need to add the following files:

### Icons (for PWA)
- `icon-72.png` (72x72)
- `icon-96.png` (96x96)
- `icon-128.png` (128x128)
- `icon-144.png` (144x144)
- `icon-152.png` (152x152)
- `icon-192.png` (192x192)
- `icon-384.png` (384x384)
- `icon-512.png` (512x512)

### Other Assets
- `favicon.ico` - Browser favicon
- `favicon.svg` - Modern SVG favicon (already created)
- `og-image.png` - Open Graph image for social media sharing (1200x630 recommended)
- `screenshot-wide.png` - Wide screenshot for PWA (1280x720)
- `screenshot-mobile.png` - Mobile screenshot for PWA (750x1334)

## Quick Icon Generation

You can use online tools to generate all icon sizes from a single source image:

1. **Favicon Generator**: https://realfavicongenerator.net/
2. **PWA Asset Generator**: https://www.pwabuilder.com/imageGenerator
3. **Figma/Photoshop**: Export at different sizes

## Recommended Logo Design

Your logo should:
- Be simple and recognizable at small sizes
- Use your brand colors (primary: #2563eb)
- Work well on both light and dark backgrounds
- Include "GD" or "GlobalDeets" text/icon

## Brand Colors

- Primary Blue: `#2563eb`
- Dark Background: `#0f172a`
- Light Background: `#f8fafc`
- Accent Purple: `#8b5cf6`
- Success Green: `#10b981`

A basic SVG favicon has been created. Generate PNG versions for full PWA support.
